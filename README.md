# Zilliqa Golang SDK
A Golang implementation of [Zilliqa](https://github.com/Zilliqa/Zilliqa) API.

The project is still under development. Please note that specifications may change greatly.

## Installation

```sh
go get -u github.com/GincoInc/zillean
```
